java 多维数组

2017-09-17
java

java的多维数组是靠一维数组实现的。

多维数组可以看作是元素为数组的数组。比如arr[][],arr[] 的元素为数组的引用，可以说arr[]的元素就是二级数组的指针。

arr的第二维数组元素的长度都是随意的。
